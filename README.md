# ShareKino

## サイト概要
映画について感想を寄せ合うサイト

### サイトテーマ
映画感想コミュニティ型サイト

### テーマを選んだ理由
映画のレビューサイトはよく目にするが、特定の映画を決めて感想をシェアしあう機会は少ないと感じたため、
新しく作ることにした。

### ターゲットユーザ
映画が好きだが、身近な人以外に感想をシェアできる人が欲しい人

### 主な利用シーン
映画鑑賞の記録
